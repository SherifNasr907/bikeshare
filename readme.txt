Introduction

This is part of Udacity Python project that uses knowledge of Python's modules- Pandas and Numpy, to make sense of the bike sharing data provided, and answer relevant questions about it.
How to use

This repository contains-
    One python file written to assess the data in the csv files- bikeshare.py.

To use the project-
    Place the bikeshare.py file and all the CSV files in the same folder.(csv files were ignored)
    Run the python file using a suitable python interpreter.
